# 哈希示例
为了巨量数据的快速存储和检索而设计的哈希表类， 可提供基于不同哈希键值对的近似常数时间的寻位存储和定位检索。  寻位和定位算法基于键对象之哈希值的每一个二进制位的状态所构造 的二叉树。未来应用于P2P网络路由和检索入口设计（SHA256）
